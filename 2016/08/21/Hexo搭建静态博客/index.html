Hexo搭建静态博客

Hexo 是一款基于 Node.js 的快速的、简单的博客框架，能够创建一个项目进行编辑后生成一套静态网页，比较适合个人博客搭建。因为 Hexo 生成的网页不依赖数据库和任何 Web 工具，所以可以把它放在 Github上，然后配置好域名以后进行访问。

1.安装前置条件

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js
  • Git

2.安装hexo

  • windows系统安装,所有带-g的都表示全局安装

  • mac安装

3.初始化博客

  • 初始博客文件夹，就是在本地创建一个文件夹叫blog

  • 安装所需模块

  • 启动服务查看基本结构,这样可以看到博客的基本样式,localhost:4000/

4.新建文章

  • 添加文章为md格式来进行编写

  • 也可以在WebStorm里创建文件,在source文件夹的_posts下,但是必须加上头信息

  • 添加完文章在启动服务

5.发布到git上

  • 下载hexo-deployer-git插件

  • 在git上创建自己的仓库,仓库名的格式如下:不然找不到 以后访问的时候也访问这个

  • 修改_config.yml,根目录下的,直接复制即可

  • 如果出现用户名不能访问的问题

  • 部署,发布

  • 以后有文章更新,可按以下步骤来进行。

    hexo generate 可以简写 hexo g
hexo deploy

    参考:https://hexo.io/
