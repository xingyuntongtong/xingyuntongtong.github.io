Mongodb 数据库

Mongoose是什么？

Mongoose是MongoDB的一个对象模型工具 同时它也是针对MongoDB操作的一个对象模型库,封装了MongoDB对文档的的一些增删改查等常用方法 让NodeJS操作Mongodb数据库变得更加灵活简单

安装mongoose

使用mongoose

Schema

  • Schema是数据库集合的模型骨架 定义了集合中的字段的名称和类型以及默认值等信息；也就是数据库里的内容，它不能操作数据库，和数据库无关只定义字段和类型,它是mongoose的一个属性
  • Schema生成Model，Model创造Entity

定义Schema

var UserSchema = new mongoose.Schema({ //UserSchema就是Schema的一个实例
  name:String, //姓名
  binary:Buffer,//二进制
  living:Boolean,//是否活着
  birthday:Date,//生日
  age:Number,//年龄
  _id:Schema.Types.ObjectId,  //主键 永远都不会变
  _fk:Schema.Types.ObjectId,  //外键 就是别人家的主键
  array:[],//数组
  arrOfString:[String],//字符串数组
  arrOfNumber:[Number],//数字数组
  arrOfDate:[Date],//日期数组
  arrOfBuffer:[Buffer],//Buffer数组
  arrOfBoolean:[Boolean],//布尔值数组
  arrOfObjectId:[Schema.Types.ObjectId]//对象ID数组
  nested:{ //内嵌文档
    name:String,
  }
},{collection:'user'});//这样指定了在数据库中就会出现user集合

Model

  • Model是由通过Schema构造而成 依赖于Schema 可以操作数据库 除了具有Schema定义的数据库骨架以外，还可以操作数据库 如何通过Schema来创建Model呢
  • 拥有了Model，我们也就拥有了操作数据库的能力

在数据库里查看的是复数 在数据库中的集合名称等于 模型名转小写再转复数,比如 User>user>users,Child>child>children

基本操作

create 增加

  • 创建文档，把参数文档保存到数据库中，是异步的 UserModel有很多方法

find 查询

  • find()返回的是一个数组，findOne()是一个对象 只找一个
  • 若没有向find传递参数，默认的是显示所有文档
  • 只要从数据库中查询出来的对象，都有一个save方法，可以把自己重新保存到数据库

remove 删除

  • justOne 如果条件匹配的记录数是多条，那么仅删除第一条 只能在命令行里用justOne
